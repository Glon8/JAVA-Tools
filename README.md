# JAVA-Tools
A collection of custom Java classes and utilities designed to simplify and speed up development.

This repository contains reusable data structures and extensions that I created for easier coding and better control over core logic.

## Contents

### Databases (Data Structures)

- **Queue** – Custom queue implementation  
- **Node** – Basic node structure  
- **BinNode** – Binary node implementation  
- **Stack** – Custom stack implementation  

### Extensions

- **Array Extension** – A custom extension of the standard array structure, providing additional functionality beyond default Java behavior  

## Purpose

- Reduce repetitive boilerplate code  
- Improve control over data structure behavior  
- Practice and strengthen core Java fundamentals
